## 概要
本の投稿、レビューアプリ

## 機能
読んだ本の写真、スコアを投稿。
投稿された内容についてのレビュー。

Firebase Authenticationでの匿名ログイン。

### 言語 / 環境
言語：React Native / DB: Firebase store, storage
